# Team Images

## Optimization Specifications

- **Max Width**: 800px ( portraits )
- **Format**: WebP (with JPEG fallback)
- **Loading**: Lazy loading (except for core team)
- **Compression**: High quality (quality 85-90% for faces)
- **Purpose**: Professional team portraits
- **Target Size**: < 100KB per image
- **Priority**: Medium

## Naming Convention
`team-{name}.{webp|jpg}`

Example: `team-jean-pierre.webp`

## Usage
- About page team grid
- Team member cards
- Author profiles
