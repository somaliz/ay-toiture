# Objets de Valeur Images

## Optimization Specifications

- **Max Width**: 1200px
- **Format**: WebP (with JPEG fallback)
- **Loading**: Lazy loading enabled
- **Compression**: High quality (quality 85-90% for detail preservation)
- **Purpose**: High-value items requiring special handling
- **Target Size**: < 200KB per image
- **Priority**: High - detail critical

## Naming Convention
`objets-valeur-{number}-{type}.{webp|jpg}`

Example: `objets-valeur-01-antique-furniture.webp`
