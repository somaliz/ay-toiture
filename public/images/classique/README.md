# Débarras Classique Images

## Optimization Specifications

- **Max Width**: 1200px
- **Format**: WebP (with JPEG fallback)
- **Loading**: Lazy loading enabled
- **Compression**: Standard (quality 80-85%)
- **Purpose**: Standard débarras operation photos
- **Target Size**: < 150KB per image
- **Priority**: Medium

## Naming Convention
`classique-{number}-{description}.{webp|jpg}`

Example: `classique-01-appartment-clearout.webp`

## Usage
- Gallery grid display
- Case study illustrations
- Service portfolio showcase
