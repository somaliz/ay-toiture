# Hero Images

## Optimization Specifications

- **Max Width**: 1920px
- **Format**: WebP (with JPEG fallback)
- **Loading**: Eager (no lazy loading) - these are above-the-fold images
- **Compression**: Aggressive (quality 75-80%)
- **Purpose**: Hero section images showcasing team in action
- **Target Size**: < 200KB per image
- **Priority**: High - critical for first impression

## Naming Convention
`hero-{number}-{location}.{webp|jpg}`

Example: `hero-01-toulon.webp`

## Usage
- Homepage hero carousel
- Category landing pages
- Service pages header images
