# Before/After Images

## Optimization Specifications

- **Max Width**: 1200px
- **Format**: WebP (with JPEG fallback)
- **Loading**: Lazy loading enabled
- **Compression**: Standard (quality 80-85%)
- **Purpose**: Transformation showcases
- **Target Size**: < 150KB per image
- **Priority**: Medium

## Naming Convention
`before-after-{location}-{before|after}.{webp|jpg}`

Example: `before-after-toulon-before.webp`

## Usage
- Transformation galleries
- Portfolio showcases
- Case study comparisons
- Social media content

## Organization
Consider pairing before/after images with sequential numbers for easy matching
